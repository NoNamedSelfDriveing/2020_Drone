# Mavlink

```
Mavlink(Micro Air Vehicle Link)는 소형 무인 장치들 및 자체내의 서로 다른 내부 컴포넌트과 통신하기 위한 프로토콜이다.
```

# Mavlink의 특징
```
1. 리틀 엔디안을 사용한다.
2. 버전이 다를경우 통신이 어긋난다.
3. 작은 메시지 버퍼를 지원한다.
4. 32bit unsigned를 지원한다.
5. IEEE 754의 float point를 지원한다.
```
# Mavlink의 패킷
```
6개의 헤더와 2개의 체크섬 그리고 데이터를 갖는 1개의 필드로 구성된다.


필드명      고유순서(바이트)        목적 및 기능
헤더1       0                      시작점을 가르키는 헤더 
헤더2       1                      페이로드 길이 값
헤더3       2                      총 패킷에서 해당하는 순서 값
헤더4       3                      발신자 시스템의 고유 ID
헤더5       4                      해당 컴포넌트 고유 ID
헤더6       5                      페이로드의 정의 ID
데이터      6(n+6)                 데이터(없을 수 있음)
체크섬1     (n+7)                  메세지 무결성 검사
체크섬2     (n+8)                  네트워크 무결성 검사
```
